Что это за окно

Пульт администратора смены. Четыре вкладки сверху:

  • 🔗 Ссылки — все ссылки на инструменты смены (педагогам, родителям, кураторам, табель, баллы, письма, паспорта) + блок «штрафников».
  • 👦 Отряды — импорт детей из CRM, разбивка по отрядам, проверка актуальности. То же окно, что в index4 — синхрон через общий лист «Дети».
  • 📒 Табель — таблица посещаемости день × ребёнок. Отметки «был / не был / договор» по каждому ребёнку и дню. Печать журнала с отметками или пустого.
  • 📋 Списки — универсальный генератор печатных имён (мини-наклейки, бирки, наклейки ФИО ×3).

Вкладка «🔗 Ссылки»

Открой эту вкладку — увидишь сетку плиток. Каждая плитка — ссылка на одно из окон системы (педагогам, кураторам, родителям и т.д.). Клик по плитке открывает окно в новой вкладке. 📋 в углу плитки — копирует URL.

Блок «штрафники» сверху над плитками: если у педагогов есть невыставленные баллы или несданные отчёты по оценочным занятиям — они появятся здесь. Раскрывается по дням смены. Красный значок «просрочено» = долги за прошлые дни или сегодняшние после 17:30. Жёлтый = «в работе до 17:30». Будущие дни свёрнуты в «📅 Заранее по будущим дням».

Вкладка «👦 Отряды»

  1. Импорт детей. Укажи группу в Альфа CRM (например 324) → ⬇ Импортировать и разбить. Подтянутся дети из филиала «Каникулы», автоматически разобьются по возрасту: младшие в отряд 1, старшие в отряд 4.
  2. Перетаскивай карточки между отрядами, если нужно поправить разбивку. Кнопка 🔒 на карточке — закрепить (защита от перезаписи при «🔀 Переразбить» или новых импортах).
  3. 🔍 Проверить актуальность — сравнивает текущий список с Альфой. Покажет кого исключили из клуба (исчезли из CRM) и кто добавился новый. Решаешь — удалить выбывших / добавить новых.
  4. 💾 Сохранить жми после ручных правок — на случай если автосохранение не успело.

Важно: это тот же экран, что у тебя в index4 (где составляется расписание). Один лист «Дети» на оба окна. Изменил тут — изменилось там. Last write wins.

Вкладка «📒 Табель»

  1. Таблица: столбец Договор + дни смены (например «1 июн, 2 июн…») + строки детей по отрядам. В конце каждого отряда есть 2 пустые строки для ручного дописывания.
  2. Ячейка кликается циклом: пусто → + (зелёный, был) → − (терракота, не был) → пусто.
  3. Столбец «Договор»: пусто → ✓ (синий, договор есть) → пусто.
  4. Автосохранение по очереди — статус «сохранение… / сохранено» показывается рядом.
  5. Печать: 🖨 Печать таблицы (с отметками) или пустой для бумажной разлиновки куратору.
«− в посещаемости» влияет на письма родителям: если ребёнок отмечен как отсутствующий в день D, то голосовой отчёт педагога за этот день НЕ попадёт в его письмо/паспорт. Это защита от «ИИ написал про действия ребёнка, которого там не было».

Вкладка «📋 Списки»

🏷 Печать имён → модалка с тремя вариантами оформления:

  • 🔖 Мини-наклейки 6×2 см — на рулон/резку, плотно без зазоров. Имя крупно, фамилия мельче.
  • 🎫 Бирки (8 на лист) — крупный шрифт, кремовый фон, терракотовая рамка. На прищепку / на бейдж.
  • 🏷 Наклейки ФИО ×3 — каждое имя повторяется три раза подряд, по 3 в ряд. Чёрная рамка, ВЕРХНИЙ РЕГИСТР.

В модалке можешь: впечатать имена руками + отметить детей галочками из списка. Соберутся в один общий список для печати. Имя везде — Фамилия + Имя (без отчества), размер шрифта автоматически подгоняется под длину имени.

🗒 Лист куратора → откроется отдельная вкладка с 4 листами A4 горизонтально, по одному отряду на лист. На каждом листе: номер и имя ребёнка (12 строк — оставлен запас под новеньких, которые приходят посреди смены), справа 5 пустых колонок без подписей — куратор ставит баллы за МК руками. Сортировка детей такая же, как в Табеле (младшие сверху).

Что важно

  • Изменения в «👦 Отряды» здесь и в index4 идут в один лист — синхрон автоматический. Не работаешь в двух окнах одновременно с одним ребёнком.
  • Перезагрузка страницы тянет данные с сервера — ничего не теряется.
  • Размер шрифта в печатных формах подгоняется автоматически (длинные имена сжимаются). Есть нижние пороги — ниже не уменьшит, иначе нечитаемо.
Не печатай таблицу посещаемости со скриншотом ссылки этой страницы — телефоны родителей видны в окне «Отряды». Если делишься скриншотами или раздаёшь распечатки команде — открывай только вкладку «📒 Табель», там телефонов нет.
