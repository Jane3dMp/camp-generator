Что это за окно

Табель работы педагогов на смене: дни × педагоги, для каждой пары — сколько занятий было, ставка, итого. Система сама считает занятия из расписания. Питание (детей × 14 ₽ за порцию) — отдельная общая колонка для сверки со столовой.

Что делать

  1. Подтяни педагогов смены. Нажми 🔄 Подтянуть из расписания — увидишь всех педагогов смены, отметь нужных и подтверди. Или добавь вручную через ➕ Добавить педагога (например, если ведёт без расписания).
  2. Подтяни занятия по дням. В каждой строке дня есть кнопка 📥 — нажми, и в строку проставятся занятия по педагогам с ставкой 35 ₽ по умолчанию. Делать это надо отдельно для каждого дня — это защита от случайной перезаписи всего табеля одним кликом.
  3. Поправь, если нужно. Ставку или количество занятий можно отредактировать вручную в любой ячейке. Если особая ячейка не должна перезаписаться при следующих «📥 Подтянуть» — нажми 🔒 рядом с ней.
  4. Отмечай выплаты галочками. «выплачено» можно отмечать по дню (все педагоги этого дня зачёркнуты) или по педагогу целиком (все его дни зачёркнуты). Как в Excel.
  5. Питание вписывается в общую колонку числом детей на дне. Сумма по столбцу — для сверки со столовой.
  6. Когда табель готов — 📥 Экспорт в Excel. Файл уходит в стандартном формате, его можно отдать бухгалтерии.
Логика подсчёта занятий: каждое МК cluster или outside засчитывается × 3 (вся группа из 3 отрядов проходит ротацию). Выезд группы (trip3_full) — тоже × 3 отряда. Выезд всех 6 (trip6) — × 6. Это всё уже зашито — считать руками не надо, просто доверяй кнопке 📥.

Куда уходит то, что ты здесь меняешь

Все правки автоматически сохраняются в Google Sheets (лист «Табель»). Индикатор ● сохранено сверху показывает статус. Бухгалтерия получает данные через файл Excel — экспорт делай вручную, когда смена закроется.

Что важно

  • 🧹 Очистить ячейки удаляет занятия и ставки, но оставляет список педагогов, питание и галочки выплат. Удобно, если ошибся при подтягивании, но не хочешь начинать с нуля.
  • 🗑 Удалить весь табель — необратимо. Удаляет всё, включая педагогов и выплаты. Двойное подтверждение защищает от случайного клика.
  • 🔒 на ячейке защищает её от перезаписи при следующих «📥 Подтянуть». «Очистить ячейки» замки игнорирует.
  • Перезагрузка страницы ничего не теряет — табель тянется с сервера при каждом открытии.
Не нажимай «🗑 Удалить весь табель», если хочешь просто переподтянуть данные. Для этого есть «🧹 Очистить ячейки» — она оставит структуру (педагогов и выплаты), а ячейки опустит. Потом нажмёшь 📥 по каждому дню.
